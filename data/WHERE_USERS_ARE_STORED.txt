Platz local accounts (email + hashed password + name)
=======================================================

FILE ON DISK (created when someone signs up):
  data/users.json

Full path on your machine:
  <this project folder>\data\users.json

WHAT IS STORED:
  - email (lowercase)
  - name (display name shown in the app)
  - passwordSalt, passwordHash (password is NOT stored in plain text — scrypt hash)
  - createdAt (ISO timestamp)

PASSWORD RESET TOKENS (forgot-password flow):
  data/password-resets.json

SECURITY:
  - These JSON files are listed in .gitignore so they are not committed to Git.
  - Keep your computer and this folder private if you use local auth.
  - Set AUTH_SECRET in .env.local for session cookies (long random string, 16+ chars).

OPTIONAL ENV (Gemini — if the API says model not found):
  GEMINI_VISION_MODEL — trash photo analysis (see https://ai.google.dev/gemini-api/docs/models )
  GEMINI_CHAT_MODEL — Civic Scout chat widget (defaults to gemini-2.0-flash)
